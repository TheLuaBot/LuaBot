# 🙋‍♂️ Contribuindo para a LuaBot

**Em primeiro Lugar, agradeço por contribuir para a LuaBot!** Isso deixa a LuaBot melhor a cada dia!

## 📚 Diretrizes
Esteja Ciente sobre as seguintes coisas:

1. Não abra Issues duplicados. Pesquise seu problema para ver se ele já foi perguntado. Problemas duplicados serão encerrados.
2. Certifique-se de fornecer informações suficientes para melhor entendimento do problema. Sem informações suficientes, **o problema pode ser insolúvel e você será solicitado a fornecer mais informações.**

### 🤖 Contribuições de IA
A LuaBot não aceita nenhum tipo de contribuição que use IA.

Usar ferramentas como Claude Code, Copilot, Gemini, ChatGPT, OpenAI Codex, etc. é simplesmente proibido. As contribuições de IA geralmente não fazem sentido e ocupam um tempo de revisão muito valioso e, portanto, são proibidas. As solicitações de pull feitas com ferramentas de IA serão fechadas instantaneamente sem revisão, não importa quão pequeno seja o conjunto de alterações.
